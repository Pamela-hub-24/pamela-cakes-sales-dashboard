
# Pamela Cakes Sales Dashboard (Power BI)

## Overview
This project presents a product-level sales analysis focused on Pamela Cakes using Power BI.

## Tools Used
- Power BI
- Power Query
- DAX

## Key KPIs
- Total Sales Revenue
- Quantity Sold
- Average Order Value (AOV)
- Gross Income
- Top Branch by Sales

## Key Insights
- Pamela Cakes is the top revenue-generating product
- Strong sales performance across multiple branches
- Consistent demand across different customer segments
